   c. `read_uploaded_file(uploaded_file) -> pd.DataFrame` — accepts a Streamlit UploadedFile object:
      - Detect file type from name (.csv vs .xlsx/.xls)
      - Read with pd.read_csv or pd.read_excel (engine="openpyxl")
      - Parse the `date` column with pd.to_datetime
      - Sort by date ascending
      - Compute derived columns to match generate_company_data() output:
        - `gross_profit = revenue - cogs`
        - `net_income = revenue - cogs - opex`
        - `month_label = date.dt.strftime("%b %Y")`
        - `month_index = range(1, len+1)` (sequential)
        - `cash_balance` — use as-is from the uploaded data (user provides actual balances)
      - Return the DataFrame with all columns that generate_company_data() produces
      - Raise ValueError with a clear message if file cannot be parsed (not for column validation — that's Phase 7)

   d. `get_template_csv_bytes() -> bytes` — returns the template CSV as bytes for st.download_button.
